## 1. Purpose

이 문서는 AI 이미지 생성 도구에 바로 넣어 쓸 수 있는 프롬프트 모음이다.

권장 사용처:

- ChatGPT 이미지 생성
- Figma AI 이미지
- Canva AI
- Adobe Express

## 2. Global Prompt Rules

모든 프롬프트에 공통으로 유지할 기준:

- cute mobile game art
- pastel colors
- soft rounded shapes
- clean silhouette
- simple composition
- no realistic photo style
- no dark background
- no text
- no watermark

### Negative Prompt Base

아래 요소는 가능하면 함께 제외:

```text
realistic, photorealistic, dark horror, gritty, noisy background, heavy shadows, overly detailed, text, letters, watermark, logo mockup, low resolution, blurry, complex background, sharp aggressive shapes
```

## 9. Figma-to-Asset Workflow

### Recommended Process

1. 먼저 Figma에서 필요한 아이콘 크기와 사용 위치를 정한다.
2. AI로 3~5개 후보를 만든다.
3. 가장 실루엣이 좋은 안을 고른다.
4. 배경 제거 또는 단색 배경 정리 후 PNG로 저장한다.
5. Figma에서 실제 UI에 넣어보고 색감과 크기를 조정한다.

### Important Rule

- 처음부터 완벽한 한 장을 기대하지 말고, `실루엣 -> 색감 -> 디테일` 순서로 개선하는 것이 빠르다.
