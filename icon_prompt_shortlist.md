# University Tycoon Icon Prompt Shortlist

## 1. Usage

이 문서는 실제 이미지 생성에 바로 넣기 위한 `짧고 강한 프롬프트`만 모아 둔 문서다.

권장 방식:

1. 프롬프트 1개로 3~4장 생성
2. 가장 실루엣 좋은 안 선택
3. 같은 프롬프트에 색상/구도만 살짝 바꿔 재생성

공통 네거티브 프롬프트:

```text
realistic, photorealistic, dark, gritty, text, letters, watermark, busy background, too much detail, blurry, low resolution, sharp aggressive shapes
```

## 4. Fast Iteration Variants

같은 프롬프트 뒤에 짧게 붙여서 결과를 바꿀 수 있는 옵션:

- `flat pastel shading`
- `slight isometric view`
- `top-down game icon`
- `extra clean silhouette`
- `more toy-like proportions`
- `soft spring palette`

## 6. Selection Rule

가장 먼저 볼 기준:

- 멀리서 봐도 무엇인지 보이는가
- 작은 크기에서 창문, 지붕, 실루엣이 뭉개지지 않는가
- 현재 앱 팔레트와 충돌하지 않는가
- 같은 세트처럼 보이는가
